# Assistant Tools Blueprint

This folder contains starter templates for the next Byte-Bot phase.

These files are not imported by the current app. They are here so you can copy them into:

```text
backend/app/services/
```

when you are ready.

Recommended copy order:

1. `tool_models.py`
2. `router_service.py`
3. `note_service.py`
4. `email_service.py`
5. `search_service.py`
6. `image_doc_service.py`
7. `chat_service.py`
8. `voice_frontend_templates.js`
9. `langgraph_service.py`

Keep the current backend stable while adding these one at a time.
